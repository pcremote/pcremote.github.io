Instructions

1) Download the App

Download the Android application from google playstore using below link.
Remotifier Android App

2) Download the Server

Download the Server for the application using the below link.
Remotifier Server

3) Run the server

Just double click the exe file then you're good to go. Optionally you can create a shortcut into your desktop or startmenu.

4) Accept the firewall

Windows will alert you to open a port. Please click on Allow Access button or ask your admin to open the port.

5) Open the Android App

Open the android app on your mobile or tablet. You can optionally turn on dark mode using the toggle button.

6) Connect with same wifi

Make sure your PC and your mobile phone are connected with same wifi network.

7) Scan the Ip

Press the scan button. If you've followed all the step's carefullly then now the app should have found the local IP address of your pc. If not please follow the below link.
Find the local Ip address of your PC.

8) Connect with your PC

Just Press the connect button to control with your windows operating system. Here all the multitasking and windows specific features are there.

9) Explore Remotifier

Now you can explore the app. Online mode let's you play videos directly into youtube within your default browser. VLC mode lets you to play your video content on VLC media player.
